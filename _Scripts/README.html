0.1 Clean the CPX data table

The initial phase of the analysis involves importing the raw Cardiopulmonary Exercise Testing (CPET) data and establishing a unique, stable identifier (ID) for each patient based on their Medical Record Number (MRN). To maintain the longitudinal integrity of the study, we filter for patients who have undergone repeated testing (at least two sessions).Furthermore, we restrict the cohort to patients with a confirmed diagnosis of Hypertrophic Cardiomyopathy (HCM) and ensure physiological effort by requiring a peak Respiratory Exchange Ratio (RER) of \(\geq 1.0\). Biometric indices, including Body Surface Area (BSA), Body Mass Index (BMI), and Lean Body Mass Index (LBMI)—the latter utilizing race-agnostic NHANES equations—are calculated for phenotypic normalization.

1 Filter by the comorbidities table

To ensure that exercise limitations are attributable to HCM-specific pathology (e.g., diastolic dysfunction or LVOT obstruction) rather than extrinsic factors, we exclude patients with secondary contributors to exercise intolerance. Patients with documented Coronary Artery Disease (CAD), Chronic Obstructive Pulmonary Disease (COPD), or Interstitial Lung Disease (ILD) are removed from the analysis.

2 Add Echocardiography Table

This step merges the most recent resting echocardiogram for each MRN, then expands the cohort filtering workflow to show how many CPX tests/patients have usable diastology variables for the serial analysis. The diastology availability flag is based on core echo measures (E/e’, LA volume, TR velocity, E/A, and deceleration time) and can be adjusted by editing diastology_vars.

5 Longitudinal Analysis

To characterize the temporal dynamics of exercise capacity in our HCM cohort, we calculated the annualized rate of change for peak oxygen consumption (\(VO_2\)). This “functional velocity” was derived from serial CPET sessions by determining the slope between the baseline and most recent follow-up tests, normalized to the FRIEND 2.0 reference equation to account for age-related shifts in predicted capacity. This approach allows for the identification of specific functional phenotypes—distinguishing “rapid progressors” from those with stable exercise capacity—which serves as the primary clinical endpoint for subsequent epigenetic association studies.

To ensure a robust longitudinal analysis, we visualized individual patient trajectories (Panel A) and the density of clinical follow-up (Panel B), ensuring sufficient temporal depth across the cohort. The heterogeneity of disease progression was quantified via the distribution of annualized \(VO_2\) deltas (Panel C). Finally, we utilized a multivariable correlation matrix (Panel D) to explore the relationships between baseline anthropometrics (including Lean Body Mass Index), ventilatory efficiency (\(VE/VCO_2\)), and the rate of functional decline.

## Clean the CPX data table

The initial phase of the analysis involves importing the raw Cardiopulmonary Exercise Testing (CPET) data and establishing a unique, stable identifier (ID) for each patient based on their Medical Record Number (MRN). To maintain the longitudinal integrity of the study, we filter for patients who have undergone repeated testing (at least two sessions).Furthermore, we restrict the cohort to patients with a confirmed diagnosis of Hypertrophic Cardiomyopathy (HCM) and ensure physiological effort by requiring a peak Respiratory Exchange Ratio (RER) of $\geq 1.0$. Biometric indices, including Body Surface Area (BSA), Body Mass Index (BMI), and Lean Body Mass Index (LBMI)—the latter utilizing race-agnostic NHANES equations—are calculated for phenotypic normalization.

# Helper: age-residualize index with nonlinear spline
residualize_on_age <- function(df, idx){
  df <- df %>% mutate(.row_id = dplyr::row_number())
  dsub <- df %>% filter(!is.na(.data[[idx]]), !is.na(age))
  if(nrow(dsub) < 10){
    df[[paste0(idx, "_resid")]] <- NA_real_
    return(df %>% select(-.row_id))
  }
  fit <- lm(as.formula(paste0(idx, " ~ ns(age, df=3)")), data=dsub)
  dsub <- dsub %>% mutate(.resid = resid(fit))
  df <- df %>% left_join(dsub %>% select(.row_id, .resid), by = ".row_id")
  df[[paste0(idx, "_resid")]] <- df$.resid
  df %>% select(-.row_id, -.resid)
}

  plots <- list(baseline = NULL, longitudinal = NULL)
  pred_grid <- function(var,df,model,lo,hi,n=200){
    g <- tibble::tibble(!!var := seq(lo,hi,length.out=n))
    ref_vars <- c("age","BMI","LBMI","VeVco2_slope","pk.RER",var)
    ref_vars <- ref_vars[ref_vars %in% colnames(df)]
    ref_vars <- setdiff(ref_vars, var)
    ref <- df %>% summarise(across(all_of(ref_vars),~median(.,na.rm=TRUE)),
                            Sex=as.character(names(which.max(table(Sex)))))
    newd <- dplyr::bind_cols(ref %>% dplyr::slice(rep(1, n)), g)
    newd$Sex <- factor(newd$Sex,levels=levels(df$Sex))
    p <- predict(model,newdata=newd,se.fit=TRUE)
    tibble(x=newd[[var]],fit=p$fit,se=p$se.fit,lo=p$fit-1.96*p$se.fit,hi=p$fit+1.96*p$se.fit,var=var)
  }

  fit_single_baseline <- function(idx){
    df <- baseline_df %>%
      filter(!is.na(.data[[outcome_var]]), !is.na(age), !is.na(Sex), !is.na(LBMI), !is.na(.data[[idx]]))
    n_df <- df %>% dplyr::summarise(n = dplyr::n()) %>% dplyr::pull(n)
    if(n_df < 10){return(NULL)}
    base <- lm(as.formula(paste0(outcome_var, " ~ age + Sex + LBMI")), data=df)
    mod <- lm(as.formula(paste0(outcome_var, " ~ age + Sex + LBMI + ns(", idx, ", df=3)")), data=df)
    lrt <- anova(base, mod)
    list(df=df, base=base, mod=mod, lrt=lrt, idx=idx)
  }

  baseline_models <- Filter(Negate(is.null), lapply(diastolic_indices, fit_single_baseline))
  baseline_lrt <- dplyr::bind_rows(lapply(baseline_models, function(x){
    tibble(
      index = x$idx,
      n = x$df %>% dplyr::summarise(n = dplyr::n()) %>% dplyr::pull(n),
      delta_AIC = AIC(x$mod) - AIC(x$base),
      p_value = x$lrt$`Pr(>F)`[2]
    )
  }))
  write.csv(baseline_lrt, paste0("../2_Output/Stats_", file_prefix, "_AllIndices_Baseline_LRT_", analysis_tag, ".csv"), row.names = FALSE)

  write.csv(broom::tidy(mod_all), paste0("../2_Output/Stats_", file_prefix, "_AllIndices_Baseline_Coefficients_", analysis_tag, ".csv"), row.names = FALSE)

  if(length(baseline_models) > 0){
    pp_list <- lapply(baseline_models, function(x){
      lo <- quantile(x$df[[x$idx]], 0.02, na.rm=TRUE)
      hi <- quantile(x$df[[x$idx]], 0.98, na.rm=TRUE)
      pred_grid(x$idx, x$df, x$mod, lo, hi) %>%
        mutate(label = label_map[var])
    })
    pp_all <- dplyr::bind_rows(pp_list)
    p_spl <- ggplot(pp_all,aes(x=x,y=fit))+
      geom_ribbon(aes(ymin=lo,ymax=hi),alpha=0.18)+geom_line(linewidth=1)+
      facet_wrap(~label,scales="free_x",nrow=2)+
      labs(title = bquote("Resting Diastolic Indices and Peak " * VO[2] * " (" * .(analysis_tag) * ")"),
           x=NULL,
           y = bquote("Predicted Peak " * VO[2] * " (" * .(outcome_label) * ")"))+
      theme_pub()
    save_pdf(p_spl, paste0("../2_Output/Figure3_Nonlinear_DiastolicIndices_", file_prefix, "_", analysis_tag, ".pdf"),12.5,6.8)
    plots$baseline <- p_spl
  }

  long_df2 <- long_df %>% mutate(time_yrs=days_from_baseline/365.25) %>% filter(is.finite(time_yrs))
  fit_single_long <- function(idx){
    df <- long_df2 %>%
      filter(!is.na(.data[[outcome_var]]), !is.na(age), !is.na(Sex), !is.na(LBMI), !is.na(.data[[idx]]))
    n_df <- df %>% dplyr::summarise(n = dplyr::n()) %>% dplyr::pull(n)
    if(n_df < 10){return(NULL)}
    base <- lmer(as.formula(paste0(outcome_var, " ~ time_yrs + age + Sex + LBMI + (1|ID)")), data=df, REML=FALSE)
    mod <- lmer(as.formula(paste0(outcome_var, " ~ time_yrs + age + Sex + LBMI + ns(", idx, ", df=3) + (1|ID)")),
                data=df, REML=FALSE)
    lrt <- anova(base, mod)
    list(df=df, base=base, mod=mod, lrt=lrt, idx=idx)
  }

  long_models <- Filter(Negate(is.null), lapply(diastolic_indices, fit_single_long))
  long_lrt <- dplyr::bind_rows(lapply(long_models, function(x){
    tibble(
      index = x$idx,
      n = x$df %>% dplyr::summarise(n = dplyr::n()) %>% dplyr::pull(n),

# Residualize all diastolic indices on age (nonlinear) using ns(age, df=3)
residualize_on_age <- function(df, idx){
  df <- df %>% mutate(.row_id = dplyr::row_number())
  dsub <- df %>% filter(!is.na(.data[[idx]]), !is.na(age))
  if(nrow(dsub) < 10){
    df[[paste0(idx, "_resid")]] <- NA_real_
    return(df %>% select(-.row_id))
  }
  fit <- lm(as.formula(paste0(idx, " ~ ns(age, df=3)")), data=dsub)
  dsub <- dsub %>% mutate(.resid = resid(fit))
  df <- df %>% left_join(dsub %>% select(.row_id, .resid), by = ".row_id")
  df[[paste0(idx, "_resid")]] <- df$.resid
  df %>% select(-.row_id, -.resid)
}

# Longitudinal Analysis

To characterize the temporal dynamics of exercise capacity in our HCM cohort, we calculated the **annualized rate of change** for peak oxygen consumption ($VO_2$). This "functional velocity" was derived from serial CPET sessions by determining the slope between the baseline and most recent follow-up tests, normalized to the FRIEND 2.0 reference equation to account for age-related shifts in predicted capacity. This approach allows for the identification of specific functional phenotypes—distinguishing "rapid progressors" from those with stable exercise capacity—which serves as the primary clinical endpoint for subsequent epigenetic association studies.

To ensure a robust longitudinal analysis, we visualized individual patient trajectories (Panel A) and the density of clinical follow-up (Panel B), ensuring sufficient temporal depth across the cohort. The heterogeneity of disease progression was quantified via the distribution of annualized $VO_2$ deltas (Panel C). Finally, we utilized a multivariable correlation matrix (Panel D) to explore the relationships between baseline anthropometrics (including Lean Body Mass Index), ventilatory efficiency ($VE/VCO_2$), and the rate of functional decline.
